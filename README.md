# Asteroids_game_in_Rust_with_Bevy
Я зробив гру Астероїди на Rust за допомогою ігрового рушія Bevy.
## Про гру Астероїди
«Астероїди» (Asteroids) — це класична відеогра 1979 року від Atariде гравець керує космічним кораблем, знищуючи астероїди та ворожі тарілки. Це також сучасна настільна гра-головоломка (Smart Games)де потрібно вивести корабель з астероїдного поля, пересуваючи блоки. 
Основні характеристики гри «Астероїди»:
-Класична відеогра (1979): Один з найуспішніших ігрових автоматів у стилі аркадного шутера. Корабель може обертатися та стріляти в астероїди, які при влучанні розпадаються на менші частини.
-Настільна гра-головоломка «Увага! Астероїди» (Asteroid Escape):
-Мета: Провести корабель через поле астероїдів,маніпулюючи космічними блоками (принцип «п'ятнашок»).
-Особливості: Включає 60 завдань різної складності (від новачків до експертів).
-Розвиток: Гра тренує логіку планування та просторове мислення.
## Про Rust
Rust — це сучасна компільована мова програмування загального призначення, орієнтована на безпеку, швидкість і паралелізм. Вона забезпечує високу продуктивність рівня C/C++, уникаючи поширених помилок роботи з пам'яттю (segfaults, витоки) завдяки унікальній системі володіння (ownership) без збирача сміття.
Де використовують Rust:
-Системне програмування: Розробка операційних систем (наприклад, Redox), браузерних рушіїв (Firefox, Servo).
-Вбудовані системи (Embedded) та IoT: Завдяки низькорівневому доступу до обладнання.
-Високонавантажені веб-сервіси: Завдяки швидкості та паралелізму.
-WebAssembly: Швидка робота в браузері.
## Про Bevy
Bevy — це безкоштовний ігровий рушій з відкритим вихідним кодом, написаний мовою програмування Rust, який вирізняється високою продуктивністю та орієнтацією на дані. Він використовує архітектуру Entity Component System (ECS) для керування ігровою логікою, рендерингом 2D/3D графіки та звуком. Основні переваги включають кросплатформенність та модульність.
Ключові особливості Bevy Engine:
-ECS (Entity Component System): Все в Bevy побудовано на основі сутностей, компонентів та систем, що забезпечує гнучкість.
-Мова Rust: Забезпечує безпеку пам'яті та високу швидкість виконання без необхідності використання збирача сміття.
-Кросплатформенність: Підтримує Windows, MacOS, Linux, Web (WASM), iOS та Android.
-Продуктивність: Сучасний рендер (2D/3D)звуковий рушій та UI-фреймворк.
-Відкритий код: Безкоштовний і розробляється спільнотою.
